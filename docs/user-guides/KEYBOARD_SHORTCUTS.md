# Keyboard Shortcuts (Desktop)

Global:
- `/` Focus search
- `g` Open outfit generator
- `Esc` Close modal/panel

Inventory:
- `j` Next item
- `k` Previous item
- `Enter` Open selected item
- `e` Edit selected item
- `l` Toggle laundry state (available <-> laundry)
- `d` Mark donate (toggle)

Bulk:
- `Shift + Click` Range select
- `Cmd/Ctrl + A` Select all in view
- `Cmd/Ctrl + Enter` Apply bulk action (when bulk modal open)

Outfits:
- `o` Open outfit board
- `p` Panic pick (generate 1 best outfit)
