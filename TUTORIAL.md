## Your First Item

Adding items to your closet is fast and friction-free‚Äîdesigned with ADHD in mind!

### Step 1: Take a Photo
1. From the home screen, tap the **+ Add Item** button (the big purple floating button)
2. Choose **Take Photo** or **Upload from Gallery**
3. Take a clear photo of your item laid flat or on a hanger
   - **Tip**: Good lighting makes AI inference more accurate
   - **Tip**: Avoid busy backgrounds if possible

### Step 2: Add Front View
- The first photo you upload is the **front view** (required)
- Once uploaded, the photo appears immediately
- Tap **Continue** to move to the next step

### Step 3: Add Back View (Optional)
- Add a back photo if your item has interesting details on the back
- **Skip this** if you're in a hurry‚Äîyou can always add it later
- Tap **Skip** or **Continue**

### Step 4: Add Label/Detail Photos (Optional)
- Upload close-ups of:
  - Brand labels (helps AI identify brand and size)
  - Care labels (washing instructions)
  - Details like embroidery, buttons, or prints
- **Progressive disclosure**: Only shown if you tap "Add More Photos"
- **Skip this** to add items quickly

### Step 5: Fill in Details (Optional)
- **Title**: Name your item (e.g., "Black Band T-Shirt")
- **Category**: AI will suggest a category, but you can change it
- **Brand**: Type the brand name if known
- **Tags**: Add tags like "goth", "comfy", "favorite", "going-out"

### Step 6: Save!
- Tap **Save Item**
- The AI will start processing your item in the background
  - Generating catalog images
  - Inferring category, colors, and attributes
  - Extracting label information (if label photos provided)
- **You can keep using the app while AI works!**

## Creating Your First Outfit

The outfit generator helps reduce decision paralysis by giving you a few curated options.

### Step 1: Start the Generator
1. From the home screen, tap **Outfits** in the navigation
2. Tap **+ Generate Outfit**

### Step 2: Set Constraints
Answer a few quick questions to guide the AI:

**Weather** (optional):
- Hot / Warm / Cool / Cold / Rain / Snow
- Tap **Auto** to use your location's current weather

**Vibe** (optional):
- **Dysphoria Safe**: Comfortable, gender-affirming pieces
- **Confidence Boost**: Your best, most empowering items
- **Dopamine**: Fun, colorful, mood-lifting pieces
- **Neutral**: No specific vibe preference

**Occasion** (optional):
- Type in where you're going (e.g., "work", "concert", "coffee date")

**Time Available**:
- **Quick**: Simple outfits, minimal accessories
- **Normal**: Includes accessories and layering

### Step 3: Generate
- Tap **Generate Outfit**
- AI creates 1-5 ranked outfit suggestions
- Each outfit shows:
  - Item thumbnails
  - Short explanation
  - Optional swap suggestions

### Step 4: Panic Pick (Emergency Mode)
If you're overwhelmed or in a hurry:
- Tap **‚ö° Panic Pick**
- AI instantly generates a single safe, reliable outfit
- No thinking required!

## Using Desktop Power Tools

On screens ‚â•1024px wide, you unlock advanced features.

### Bulk Edit Table
**What it does**: Edit multiple items at once

**How to use**:
1. Switch to **Table View** (toggle in top right)
2. Select multiple items using checkboxes
3. Use bulk actions:
   - Change state (Available ‚Üí Laundry)
   - Add tags to multiple items
   - Delete multiple items
   - Batch regenerate AI images

**ADHD benefit**: Reduce repetitive clicks, batch similar tasks

### Keyboard Shortcuts
Press `?` or `Shift+/` to see all shortcuts:

- **Navigation**:
  - `H`: Home
  - `A`: Add Item
  - `O`: Outfits
  - `S`: Settings
- **Actions**:
  - `Esc`: Close dialogs, cancel actions
  - `Enter`: Confirm primary action
  - `/`: Focus search
- **Bulk Edit**:
  - `Ctrl/Cmd + A`: Select all
  - `Delete`: Delete selected items

### Side-by-Side Compare
Compare two items before deciding:
1. Select two items in table view
2. Click **Compare** button
3. View items side-by-side with full details

### Advanced Search
Click **More Filters** to access:
- Search by brand
- Filter by multiple tags
- Search by color palette
- Combine multiple filters

## Common Workflows

### Morning Routine: Get Dressed Fast
1. Tap **Outfits** ‚Üí **Panic Pick**
2. Get one instant outfit suggestion
3. Wear it and rate it later

### Weekly Laundry Day
1. Go to **Home**
2. Tap category **All**
3. Switch to **Table View** (desktop)
4. Select all items you just washed
5. Bulk action: **Set State ‚Üí Available**

### Declutter Session
1. Go to **Analytics** to see items by category
2. Identify overcrowded categories
3. Go to **Home** ‚Üí **Table View**
4. Select items you haven't worn
5. Bulk action: **Set State ‚Üí Donate**
6. When ready, take donation items in one trip

### Plan Outfits for the Week
1. Go to **Outfits** ‚Üí **Generate**
2. Set **Weather** for the week
3. Generate 5-7 outfits
4. Screenshot or save each
5. Rate them after wearing

### Tag Organization
1. Go to **Analytics** to see your top tags
2. Notice duplicate tags (e.g., "black" vs "black clothes")
3. Go to **Home** ‚Üí **Advanced Search**
4. Search for old tag ‚Üí Select all
5. Remove old tag
6. Add new consistent tag

## Tips for Success

### ADHD-Friendly Habits
- **Add items immediately**: When you buy something new, photograph it right away
- **Use Panic Pick**: No shame in letting AI decide
- **Set laundry reminders**: Enable in Settings
- **Batch similar tasks**: Use bulk edit for repetitive actions
- **Progressive disclosure**: Don't fill in every detail‚ÄîAI can help

### Photography Tips
- **Lay items flat** on a neutral surface
- **Use natural light** near a window
- **Take photos in batches** when energy is high
- **Don't aim for perfection**‚ÄîAI cleans up the image

### Tagging Strategy
Use tags for what matters to you:
- **Style**: goth, punk, emo, soft, edgy
- **Comfort**: comfy, stiff, scratchy, soft
- **Occasion**: work, concert, casual, fancy
- **Mood**: confidence, dysphoria-safe, dopamine
- **Functional**: layering, statement-piece, basics
