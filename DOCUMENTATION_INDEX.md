## Using MkDocs

MkDocs provides a better reading experience with:
- ‚úÖ Search functionality
- ‚úÖ Clean navigation
- ‚úÖ Dark mode support
- ‚úÖ Mobile-friendly
- ‚úÖ ADHD-friendly layout

### Install and Run

```bash
# Install MkDocs (one time)
pip install mkdocs-material

# Serve documentation locally
mkdocs serve

# Build static site
mkdocs build
```

### Deploy to GitHub Pages

```bash
mkdocs gh-deploy
```

This builds the docs and pushes to `gh-pages` branch for hosting at:
`https://shelbeely.github.io/ADHD-Closet/`
